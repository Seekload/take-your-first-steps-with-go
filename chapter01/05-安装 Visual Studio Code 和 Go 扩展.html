安装 Visual Studio Code 和Go扩展

安装 Visual Studio Code 和 Go 扩展

可以使用任何文本编辑器来编写 Go 应用程序，但使用 Visual Studio Code 等集成开发环境 (IDE) 时，可以更快地编写代码。 如果没有 IDE，则必须从头开始编写每行。 刚开始时，这有助于熟悉 Go 语法；但从长远来看，这样做可能需要花费更长的时间才能编写应用程序。

因此，建议一开始就使用 Visual Studio Code，它是免费的开放源代码工具，并且提供 Go 扩展来帮助你更快编写代码。

使用 Visual Studio Code 和 Go 扩展有许多好处。 方法：

  • 按照 Go 标准格式化应用程序。
  • 添加新行并保存代码后立即添加包引用。
  • 删除应用程序不再使用的包引用。
  • 通过设置断点并在运行时使用观察程序来调试应用程序。
  • 获取代码和包引用的建议完成项。
  • 使用悬停信息连接到代码文档。
  • 获取签名帮助，如列出函数的参数。
  • 即使有多个文件，也可以轻松地在项目代码中导航。
  • 登记对测试的支持，如为函数生成测试框架。

接下来分别讲下在 Linux、MacOS 和 Windows 下如何安装 Visual Studio Code 和 Go 扩展。

安装 Visual Studio Code

Linux

步骤 1：下载 Visual Studio Code 安装程序

在“Visual Studio Code”页上，选择“Linux”框，然后选择“.deb”或“.rpm”选项。 你的选择取决于 Linux 分发是使用基于 Debian (DEB) 还是基于 Red Hat 包管理器 (RPM) 的包管理器。 几秒钟后，系统会提示你将文件保存在本地。

步骤 2：启动 Visual Studio Code 安装程序

找到并打开刚刚下载的文件。 按照说明在工作站上安装应用。

步骤 3：启动 Visual Studio Code

安装完成后，打开 Visual Studio Code 以确认它是否正常工作。

MacOS

步骤 1：下载 Visual Studio Code

在“Visual Studio Code”页上，选择“Mac”框以下载应用。 几秒钟后，系统会提示你将文件保存在本地。

步骤 2：将应用移到“应用程序”文件夹

找到刚刚下载的 Visual Studio Code 文件，然后将其拖到查找器中的“应用程序”文件夹。

步骤 3：从“应用程序”文件夹启动应用

从“应用程序”文件夹中，双击“Visual Studio Code”图标以启动应用。

备注：

你可能会看到一个警告消息，指示无法打开 Visual Studio Code，因为 Apple 无法检查它是否为恶意软件。 如果是这样，请执行以下操作：

  • 选择“确定”关闭消息。
  • 在“应用程序”文件夹中，右键单击（或按住 Control 并单击）Visual Studio Code 文件，然后选择“打开”。

如果选择此选项，则 Visual Studio Code 应打开，而不会出现任何其他问题。

或者，可以使用 APT（高级包工具）或 YUM（Yellowdog 更新程序修订版）等安装 Visual Studio Code。

Windows

步骤 1：下载 Visual Studio Code 安装程序

在“Visual Studio Code”页上，选择“Windows”框。 几秒钟后，系统会提示你将文件保存在本地。

步骤 2：启动 Visual Studio Code 安装程序

找到并打开刚刚下载的文件。 按照说明在工作站上安装应用。

步骤 3：启动 Visual Studio Code

安装完成后，打开 Visual Studio Code 以确认它是否正常工作。

安装适用于 Visual Studio Code 的 Go 扩展

适用于 Visual Studio Code 的 Go 扩展有助于更有效地编写 Go 应用。 你将了解为什么要使用 IDE 来开发 Go 应用。

步骤 1：打开“扩展”视图

打开 Visual Studio Code，然后执行以下操作之一：

  • 选择“视图”，然后选择“扩展”。
  • 在左侧工具栏中，选择“扩展”图标。

步骤 2：搜索 Go 扩展

在“在市场中搜索扩展”框中，输入“Go”。

选择“Google Go 团队”发布的“Go”扩展（通常是列表中的第一个）。

扩展详细信息显示在右侧。

步骤 3：安装 Go 扩展

在扩展说明下，选择绿色的“安装”按钮，将其安装到工作站上。

安装完成后，按钮标签更改为“卸载”，这表示已成功安装扩展。
