知识检查

请回答以下问题，展示你学到的知识：

1.是否可以在 if 语句外使用 if 语句中声明的变量？

  • 可以，因为变量是全局作用域变量。
  • 不可以，变量作用域仅限 if 块或嵌套式 if 语句。
  • 可以，前提是在函数中使用它们。
  • 不可以，变量必须在 if 语句外部声明。

2.在 switch 块中，是否需要在每个 case 语句末尾包含 break 关键字？

  • 否，Go 不要求包含 break 关键字。
  • 是的，否则会对以下 case 语句进行评估。
  • 否，Go 不支持 break 关键字。

3.在 Go 中，panic() 函数的作用是什么？

  • 它会引发需要捕获的错误。 否则，程序会崩溃，并且不会显示任何错误。
  • 它会将紧急消息输出到控制台。
  • 它会导致程序崩溃，并输出错误消息及其堆栈跟踪。
